**Feature #51:** Implement Seamless Infinite Loop for Slider

**Category:** Slider

**Description:** When in slider mode, reaching the last slide causes the entire slider to snap back to the beginning. Instead, the slider should loop continuously so that after the last slide, the first slide appears seamlessly as if it's the next slide in sequence (true infinite scrolling in the same direction).

### What's Needed for Seamless Loop

To implement true seamless infinite scrolling, the following changes are required:

1. **Clone Slides:**
   - Clone the first `slidesToShow` slides and append to the end
   - Clone the last `slidesToShow` slides and prepend to the beginning
   - Example: If showing 3 slides, clone slides 1-3 to end, slides 10-12 to beginning

2. **Update Transform Logic:**
   - Allow the slider to scroll past the actual end
   - When reaching a cloned slide, instantly (during transition) reset position to the real slide
   - Use CSS transitions for smooth movement, disable transition for instant resets

3. **Handle Edge Cases:**
   - Update dots/pagination to reflect real slide position
   - Ensure autoplay works correctly with clones
   - Handle responsive resize (need to recreate clones if `slidesToShow` changes)
   - Fix touch swipe handling with cloned slides
